Bug Bounty Challenge

Welcome to the Bug Bounty Challenge. This challenge is designed to simulate real-world bug bounty programs where you will attempt to find vulnerabilities in a controlled environment.

Objective

Your goal is to identify security flaws in the provided web application. You can use tools such as Burp Suite, OWASP ZAP, or manual testing to uncover vulnerabilities.

Rules

  • No destructive testing (e.g., denial of service).
  • Report vulnerabilities in a responsible manner.
  • Follow ethical guidelines throughout the challenge.

Tools Provided

  • Access to the vulnerable web application
  • Burp Suite Community Edition
  • OWASP ZAP
