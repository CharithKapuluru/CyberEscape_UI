Available Challenges

Bug Bounty Programs

  • Web Application XSS Hunt: Identify cross-site scripting (XSS) vulnerabilities in a sample e-commerce site.
  • SQL Injection in Login Form: Find and exploit SQL injection vulnerabilities in the login mechanism of a web application.
  • API Rate Limiting Test: Test a public API for rate limiting vulnerabilities and submit a report.
  • Open Redirect Challenge: Identify open redirect issues in a URL redirector web app.
  • Mobile App Data Leakage: Analyze a mobile app for sensitive data exposure and report the issues found.

Vulnerability Challenges

  • Reverse Engineering Challenge: Download the provided binary file and reverse engineer it to discover hidden vulnerabilities.
  • Web Application Bug Bounty: Test your skills on a vulnerable web application and report the bugs you find.
  • API Security Challenge: Discover vulnerabilities in our simulated API and practice API pentesting.
  • Buffer Overflow in C Program: Analyze a C program and identify buffer overflow vulnerabilities.
  • Privilege Escalation on Linux: Gain root access by exploiting misconfigurations on a Linux virtual machine.

Code Auditing Examples

  • JavaScript XSS Prevention: Audit a JavaScript codebase to find and fix XSS vulnerabilities.
  • Python Insecure Deserialization: Analyze a Python script for insecure deserialization vulnerabilities and suggest improvements.
  • SQL Injection in PHP Code: Review a PHP codebase to identify and fix SQL injection issues.
  • Hardcoded Secrets in Java: Audit a Java application for hardcoded secrets and propose secure alternatives.
  • Access Control in Node.js: Analyze a Node.js application to ensure proper access control mechanisms are in place.
